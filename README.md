# Kit Maker

Create beautiful keep-in-touch cards to share your contact info before parting ways.

**[Live Demo](https://bang9.github.io/kit-maker/)**

## Features

- Background image upload with drag & drop
- Overlay color & opacity controls
- Date range picker
- Emoji picker
- Contact info (Phone, Email, KakaoTalk, Instagram, LinkedIn)
- Live preview with mobile-friendly thumbnail
- Download as PNG image

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Zustand
- shadcn/ui

## Getting Started

```bash
pnpm install
pnpm dev
```
